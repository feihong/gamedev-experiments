# Raylib quickstart

It is amazingly easy to get started with raylib, but unicode support seems really bad at the moment.

## Installation

```
brew install raylib
```

## Commands

Compile and run

    cc hello.c -o hello `pkg-config --libs --cflags raylib` && ./hello
